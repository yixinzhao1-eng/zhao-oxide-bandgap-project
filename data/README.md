# Data Files

This directory contains the raw project dataset, the processed MAGPIE modeling
dataset, and the CSV outputs generated by the modeling and visualization
notebooks.

## Main Datasets

### Raw Project Dataset

`zhao_nonmetal_oxide_bandgap_project.csv`

Generated by:

`notebooks/01_data_acquisition.ipynb`

It contains:

- 40,078 non-metallic oxygen-containing inorganic materials
- 22,816 unique reduced-formula groups
- DFT-calculated band gaps
- Crystal-system labels
- Formation energy per atom
- Energy above the convex hull
- Density
- Cell volume
- Number of atomic sites
- Number of elements
- Reduced formula for grouped validation

### Processed MAGPIE Dataset

`zhao_nonmetal_oxide_bandgap_magpie_project.csv`

Generated by:

`notebooks/02_eda_featurization.ipynb`

It contains:

- 40,078 materials
- 132 MAGPIE composition descriptors
- DFT-calculated band gap as the target
- Reduced formula as the GroupKFold grouping variable
- Crystal-system labels
- Six numerical stability and structure metadata features

The API filter creates a broad oxygen-containing dataset and may include
multianion materials. It is therefore broader than a strict binary-oxide-only
dataset.

## Reproduction Workflow

Run the notebooks in this order:

1. `notebooks/01_data_acquisition.ipynb`
2. `notebooks/02_eda_featurization.ipynb`
3. `notebooks/03_modeling.ipynb`
4. `notebooks/04_results_visualization.ipynb`

## API Key and Privacy

No Materials Project API key is stored in this directory or in any CSV file.

The API key must be supplied locally through a private `.env` file in the
repository root. The real `.env` file must not be committed to GitHub.

## GitHub File-Size Note

The raw and processed datasets may be omitted from GitHub if they exceed the
repository or course file-size limits. They can be regenerated by running the
data-acquisition and featurization notebooks in order.
